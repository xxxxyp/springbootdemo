AOP核心概念：
1、横切关注点
	对哪些方法进行拦截，拦截后怎么处理，这些关注点称之为横切关注点
2、切面（aspect）
	类是对物体特征的抽象，切面就是对横切关注点的抽象
3、连接点（joinpoint）
	被拦截到的点，因为spring只支持方法类型的连接点，所以spring中连接点
	指的就是被拦截到的方法，实际上连接点还可以是字段或构造器
4、切入点（pointcut）
	对连接点进行拦截的定义
5、通知（advice）
	所谓的通知就是指拦截到连接点之后要执行的代码，通知分为前置，后置，异常
	，最终，环绕五类
6、AOP代理，目标对象
	代理的目标对象，分为JDK动态代理和CGLIB代理，前者基于接口，后者基于子类
7、织入（weave）
	将切面应用到目标对象并导致代理对象创建的过程
8、引入（introduction）
	在不修改代码的前提下，引入可以在运行期为类动态的添加一些方法或字段
场景：权限认证、日志、事物、参数验证等。
